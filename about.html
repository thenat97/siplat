About us

We are a team of young students that want to help out others, the project is very simple, but we hope it helps someone out there. We believe that people should have powerful and usefull tools, that is why this project is free and open source.


We also care about the privacy of others, and that is why we don't recollect data of any type, all of the data is yours and stays with you, you can check out the code of the application in our Github repository.


What is next?

The application is far from being complete, so we we will continue developing this project in order to add new funcionalities that help out the students

How can I help?

Since we don't charge for the use of the application or from gathering information, our only way of getting resources is from donations. So please consider donating to support this project.
